## 3. Innovations in the Claude Platform: Speed, Scale, and Outcomes
▶ [34:02–53:22](https://www.youtube.com/watch?v=GMIWm5y90xA&t=2042s)
Angela and Katelyn introduce new features on the Claude platform designed to help businesses harness AI's exponential power, addressing the two main challenges: achieving the right outcomes and shipping fast at scale.
One solution is the **Advisor Strategy**, which allows smaller, cheaper models (like Haiku or Sonnet) to execute tasks while consulting a larger, more intelligent model (like Opus) for advice, leading to frontier model quality at significantly reduced costs (e.g., Eve Legal achieving 5x lower cost).
To boost speed and scale, Anthropic introduces **Claude Managed Agents**, an agentic harness with production-grade infrastructure that bundles best practices like memory management. This enables teams to move from prototype to production 10 times faster, as demonstrated by Notion.
Further enhancements to Managed Agents include:
*   **Multi-agent orchestration**: Creating fleets of agents to tackle complex tasks with independent contexts.
*   **Outcomes**: Allowing users to specify success criteria via a rubric, with a grader agent ensuring iterations until these are met.
*   **Dreaming**: An exciting self-learning capability where Claude inspects past sessions, identifies missed skills, and applies learnings to its memory autonomously.
A live demo showcases these features with "Lumara," a fictional startup using agents to autonomously land drones on the moon, demonstrating how dreaming improved simulation success rates from four to six out of six sites with a single button press.

## 4. Revolutionizing Software Development with Claude Code
▶ [53:22–1:15:43](https://www.youtube.com/watch?v=GMIWm5y90xA&t=3202s)
Cat and Boris highlight Claude Code's mission to bridge the gap between developer ideas and market-ready products by eliciting frontier intelligence from models. They describe the evolution of developer interaction, from extensive hand-holding to delegating permissions and reviewing pull requests from autonomous agents. Claude Code offers various interfaces—CLI for power users, IDE for integrated code changes, and the new Desktop app for a full-screen graphical experience with built-in previews and agent control.
The internal adoption at Anthropic has led to a 200% increase in PRs per engineer while maintaining code quality. New primitives and automations on Claude Code address common developer pain points:
*   **Code review**: Agents catch critical bugs.
*   **Remote control**: Enables coding on the go via mobile apps.
*   **Auto-fix**: Proactively resolves flaky CI tests, code review comments, and merge conflicts.
*   **Routines**: Configurable automations that kick off Claude Code tasks based on webhooks, API events, or schedules.
*   **Claude Security**: Scans codebases for vulnerabilities and uses Claude Code to address them.
These tools have been adopted by companies like Shopify (across engineering and non-engineering teams) and Mercado Libre (23,000 engineers, 500,000+ PRs reviewed, 9,000+ apps modernized), empowering managers and VPs to engage with coding again.
A live demo illustrates Claude Code Desktop, showing an engineer at "Acme Pay" adding refund functionality, where Claude autonomously writes, verifies, and fixes a race condition, and then manages parallel sessions and uses routines for asynchronous automation and CI auto-fix.
