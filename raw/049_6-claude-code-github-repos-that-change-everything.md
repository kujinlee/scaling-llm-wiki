## 2. Superpowers: The Core Development Methodology
▶ [0:38–5:42](https://www.youtube.com/watch?v=L2JKgj7WzU4&t=38s)
Superpowers is a comprehensive development methodology, not just a prompt pack, that significantly enhances Claude Code. It includes skills, agents, hooks (rules), and configuration files. Before writing any code, it forces a conversation to understand the project, generates detailed specifications, plans for 2-5 minutes, and implements a red-green-refactor TDD approach. It dispatches sub-agents to work in parallel, reviews its own code, and handles branch completion and pushing to GitHub. Installing Superpowers transforms Claude from a basic autocomplete into a junior engineer with immense discipline. It can be installed directly as a plugin via `/plugin install superpowers` or by cloning the GitHub repo (github.com/obra/superpowers).
