# Build an OKF brain like mine!

**Channel:** Marie Haynes  | **Duration:** 22:48 | **URL:** https://www.youtube.com/watch?v=esYAIA6lU-s

> [!summary] Quick Reference
> **TL;DR:** This video demonstrates building a personal knowledge system with Google's Open Knowledge Format (OKF) for efficient information management and AI-driven automation.
>
> **Key Takeaways:**
> - OKF structures knowledge using markdown and YAML front matter for AI readability.
> - Use OKF playbooks to automate complex tasks, significantly increasing your productivity.
> - OKF allows AI agents to intelligently navigate knowledge bases, saving on token usage.
> - Maintain accuracy by keeping a "human-in-the-loop" when AI agents propose OKF updates.

## 1. Introduction to Google's Open Knowledge Format (OKF)
▶ [0:00–2:10](https://www.youtube.com/watch?v=esYAIA6lU-s&t=0s)
The video introduces the speaker's personal "brain" built using Google's Open Knowledge Format (OKF). OKF is presented as a standardized structure for knowledge management, allowing AI agents to easily read and interact with information. The speaker highlights how even non-coders can leverage language models to create similar systems.

---

## 2. Understanding OKF Structure and Components
▶ [2:10–6:14](https://www.youtube.com/watch?v=esYAIA6lU-s&t=130s)
The core of OKF relies on markdown files with a standardized structure, particularly the YAML front matter. This front matter contains essential metadata like `type`, `title`, `description`, and `tags`, enabling agents to quickly understand the file's content and connect related information. The speaker uses types such as concepts, playbooks, references, and systems to categorize knowledge.

---

## 3. Practical Applications of OKF: Beyond Simple RAG
▶ [6:14–11:09](https://www.youtube.com/watch?v=esYAIA6lU-s&t=374s)
OKF offers a more efficient alternative to traditional Retrieval Augmented Generation (RAG) by allowing agents to intelligently navigate a structured knowledge base, saving on token usage. Playbooks are demonstrated as powerful tools for automating complex tasks like generating client proposals or conducting detailed site impact analyses after Google updates, significantly boosting productivity.

---

## 4. Building and Interacting with Your OKF Brain
▶ [11:09–17:37](https://www.youtube.com/watch?v=esYAIA6lU-s&t=669s)
The video demonstrates a visual knowledge graph interface where each node represents a markdown file. The agent proposes actions like creating new references or updating concepts when new information is ingested, requiring human permission. This "human-in-the-loop" approach ensures accuracy and relevance, allowing the OKF to intelligently break down complex information into interconnected concepts rather than just storing entire documents.

---

## 5. Demonstrating OKF Ingestion and Querying
▶ [17:37–19:47](https://www.youtube.com/watch?v=esYAIA6lU-s&t=1057s)
The speaker shows a live example of ingesting a new Google documentation link. The agent processes the link, proposes a plan to create a new reference, extracts key features, and updates related existing concepts within the OKF. Subsequently, the brain is queried to generate a concise summary of the newly ingested information, showcasing its utility for rapid information retrieval and report generation.
