## 1. The Need for Agent-Centric Tooling
▶ [1:45–8:32](https://www.youtube.com/watch?v=wI0ptqCSL0I&t=105s)
The proliferation of smarter AI models means developers are increasingly spending time babysitting agents or acting as glorified QA testers. Traditional software tooling was built with humans in mind, but now agents write much of our code. This shift necessitates reconsidering our toolchain to better support AI agents. Key prerequisites for advanced Cloud Code usage include maintaining high-quality CloudMD, integrating essential developer tools like Slack or DataDog, and setting up remote environments on Cloud Code web to decouple compute from local machines. These foundational steps enhance the AI's context and performance.
