Unmasking Malicious Skills: Six Categories of Attacks (1:22–4:02) dig deeper ▾

Six attack categories target AI agent skills, from hidden code to credential theft, while basic scans catch only pattern-based threats.

  • Malicious code can be embedded as hidden instructions in comments, invisible characters, or scrambled text readable only by AI.
  • Attackers create tools with names identical to trusted agent tools by using homoglyphs, facilitating impersonation.
  • Skills may lie about functionality, such as a formatter secretly accessing the internet or a reader writing files and running commands.
