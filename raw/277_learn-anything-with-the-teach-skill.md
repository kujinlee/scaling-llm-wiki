## 1. Introducing the Stateful "Teach" Skill and its Philosophy
▶ [0:00–2:47](https://www.youtube.com/watch?v=s5T5oQJcJ6U&t=0s)
Speaker's 10 years of teaching experience led to developing the "Teach" skill, designed to make learning anything accessible and personalized. It successfully helped the speaker learn to solve a Rubik's Cube. The core design principle is that "Teach" is a stateful skill, meaning it retains memory of past interactions and learning progress, unlike stateless skills which do not. This stateful nature allows it to act like a real teacher, remembering what a student has learned and tailoring future lessons.

---

## 2. Core Learning Components and Adaptive Lesson Generation
▶ [2:47–4:17](https://www.youtube.com/watch?v=s5T5oQJcJ6U&t=167s)
Upon initiation, the skill first establishes a `mission.md` file to understand the student's "why" for learning. It then generates a set of high-trust, primary source resources by searching the web. Lessons are created as interactive HTML files, stored in a `lessons` folder, offering a rich, expressive, and engaging learning experience with diagrams, simple explanations, call-outs, and quizzes. These lessons are designed to be concise and focused on immediate learning needs.

---

## 3. Tracking Progress and Applying the Zone of Proximal Development
▶ [4:17–5:22](https://www.youtube.com/watch?v=s5T5oQJcJ6U&t=257s)
The "Teach" skill maintains `learning records` to track the student's progress and reported understanding, allowing it to dynamically tailor subsequent lessons. A key pedagogical concept employed is the "Zone of Proximal Development" (ZPD), ensuring lessons are perfectly challenging without being intimidating. This adaptive approach means the skill creates bespoke lessons, like a specialized one for a specific Rubik's Cube algorithm, incorporating interactive elements for practice, maintaining optimal engagement and learning pace.

---

## 4. Supporting Resources and Fostering Community Wisdom
▶ [5:22–9:58](https://www.youtube.com/watch?v=s5T5oQJcJ6U&t=322s)
To support comprehensive learning, the skill generates reference materials such as a `glossary.md` for jargon and `cheat sheets` (e.g., a Rubik's Cube solve card) for quick access to key information. It also creates an internal `notes.md` for the agent to track user preferences and watch-outs. Critically, for acquiring "wisdom," the skill encourages interaction with learning communities, guiding learners to external forums for deeper engagement and real-world application of their knowledge and skills, rather than keeping them reliant on the agent for all learning.

## Conclusion
▶ [9:58–13:05](https://www.youtube.com/watch?v=s5T5oQJcJ6U&t=598s)
The "Teach" skill offers significant practical applications, from streamlining developer onboarding to new codebases by providing personalized, contextual learning, to empowering individuals to learn new personal skills and hobbies like chess openings. This tool highlights the unique position of developers as early adopters and builders in the age of AI, demonstrating how AI can be leveraged to create sophisticated, adaptive learning experiences that extend beyond coding into diverse domains, preparing users for a future where working with AI is ubiquitous.
