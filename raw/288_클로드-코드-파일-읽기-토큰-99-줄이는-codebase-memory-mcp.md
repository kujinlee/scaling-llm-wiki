## 1. 코드 파일 읽기 토큰 문제 해결
▶ [0:10–1:34](https://www.youtube.com/watch?v=QOAJYkuX_-Y&t=10s)
AI가 코드 파일을 읽는 과정에서 발생하는 막대한 토큰 비용 문제를 해결하는 코드베이스 메모리 MCP 도구를 소개합니다. 기존에는 AI가 코드 구조를 파악하기 위해 수십 개의 파일을 일일이 열어보면서 토큰을 소모했습니다. 이는 지도가 없는 도시에서 가게를 찾기 위해 집집마다 문을 두드리는 것에 비유될 수 있습니다. 이 도구는 이러한 비효율성을 99%까지 줄여줍니다.

## 2. 코드베이스 메모리 MCP란?
▶ [1:34–2:13](https://www.youtube.com/watch?v=QOAJYkuX_-Y&t=94s)
코드베이스 메모리 MCP(Codebase Memory MCP)는 여러분의 전체 코드를 미리 분석하여 "지식 그래프" 또는 "코드 지도"를 생성하는 도구입니다. 어떤 함수가 어디에 있고, 무엇이 무엇을 호출하며, 어떤 화면 주소로 연결되는지 등 코드의 구조를 지도처럼 정리해 줍니다. 158개 언어를 지원하며, 가벼운 실행 파일 하나로 구성되어 있습니다. 이 도구 자체에는 AI가 포함되어 있지 않으며, 생성된 지도는 클로드 코드(Claude Code)와 같은 AI가 해석하여 사용합니다.

---

## 3. 놀라운 토큰 절감 효과 및 작동 방식
▶ [2:13–3:53](https://www.youtube.com/watch?v=QOAJYkuX_-Y&t=133s)
이 도구를 사용하면 토큰 절감 효과가 매우 뛰어납니다. 개발사의 측정 결과에 따르면, 코드 구조 관련 질문 다섯 개를 처리할 때 기존 방식으로는 약 41만 2천 토큰이 소모되었지만, 코드 지도를 활용했을 때는 약 3천 4백 토큰으로 120배(99.2%)가량 줄어들었습니다. MCP는 AI에 외부 도구를 연결하는 표준 단자와 같아서, 코드 지도는 클로드 코드에 쉽게 연결됩니다. 연결 후 AI가 코드를 검색할 때 이 도구가 중간에서 코드 지도의 관련 부분을 함께 제공하여, 사용자가 별도의 명령 없이도 AI가 파일을 직접 읽기 전에 지도에서 답을 찾도록 돕습니다.

## 4. 설치 및 사용 방법
▶ [3:53–5:18](https://www.youtube.com/watch?v=QOAJYkuX_-Y&t=233s)
설치는 매우 간단합니다. 터미널에 한 줄 명령어를 입력하는 것으로 완료되며, 이 과정에서 클로드 코드와의 연결 및 필요한 모든 설정이 자동으로 이루어집니다. 설치 후에는 클로드 코드에게 "이 프로젝트 인덱싱 해 줘"라고 말하면, 도구가 자동으로 코드 지도를 생성하기 시작합니다. 인덱싱 작업은 대부분의 프로젝트에서 신속하게 완료되며, 한 번 생성된 지도는 계속 보관되어 대화를 새로 시작하거나 컴퓨터를 재시작해도 유지됩니다. 이는 AI가 같은 프로젝트를 반복해서 파악하느라 토큰을 낭비하는 것을 막아줍니다.

---

## 5. 다른 AI 코딩 도구와의 시너지 및 한계
▶ [5:18–6:08](https://www.youtube.com/watch?v=QOAJYkuX_-Y&t=318s)
코드베이스 메모리 MCP는 RTK(터미널 출력 절감)나 세레나(필요한 코드 부분만 읽게 함)와 같은 다른 AI 코딩 도구들과 충돌 없이 함께 사용할 수 있습니다. 각 도구가 토큰을 절감하는 영역이 다르기 때문입니다. 하지만 이 도구의 한계점도 명확합니다. 지도를 그리는 역할만 하며, 해석과 판단은 전적으로 AI의 몫입니다. 또한, 코드의 구조를 기반으로 지도를 생성하므로 실행 중에 동적으로 결정되는 정보는 지도에 포함되지 않을 수 있습니다. 아직 나온 지 오래되지 않은 도구이므로, 중요한 작업에는 결과를 확인하며 사용하는 것이 좋습니다.

▶ [6:08–6:43](https://www.youtube.com/watch?v=QOAJYkuX_-Y&t=368s)
코드베이스 메모리 MCP는 AI가 코드 파일을 일일이 읽지 않고도 효율적으로 작업할 수 있도록 돕는 무료 오픈소스 도구입니다. 큰 프로젝트의 AI 코딩 요금 부담을 줄이고 효율성을 높이려는 개발자에게 특히 유용합니다. 간단한 설치와 사용법으로 높은 토큰 절감 효과를 기대할 수 있습니다.
